# AI vs Human Text Detection

## Project Overview
This project compares three deep learning models to classify whether a piece of text is written by a human or generated by AI.

## Models Used
| Model | Type |
|---|---|
| Logistic Regression | Baseline (TF-IDF) |
| LSTM | Sequential Deep Learning |
| BERT | Transformer-based |

## Dataset
- Source: Kaggle - AI vs Human Text Dataset
- Binary labels: `0` = Human, `1` = AI

##  Project Structure
ai-vs-human-text-detection/
├── dataset/       → Dataset used for training and testing
├── notebooks/     → One Jupyter notebook per model
├── results/       → Final comparison charts and metrics
└── README.md      → Project documentation
## How to Run
1. Open any notebook in the `notebooks/` folder
2. Click **"Open in Colab"**
3. Run all cells from top to bottom
