# 🤖 AI vs Human Text Detection

## 📌 Project Overview
This project compares three deep learning models to classify whether a piece of text is written by a human or generated by AI. The models are trained and evaluated on a large dataset containing both human-written and AI-generated text.

## 🧠 Models Used
| Model | Type | Description |
|---|---|---|
| Logistic Regression | Baseline | Uses TF-IDF features to classify text |
| LSTM | Deep Learning | Learns patterns from sequences of words |
| BERT | Transformer | Captures deep contextual meaning in language |

## 📊 Dataset
- Source: [AI vs Human Text - Kaggle](https://www.kaggle.com/datasets/shanegerami/ai-vs-human-text)
- Total samples: 487,235
- Binary labels: `0` = Human, `1` = AI generated
- Dataset is not stored in this repo due to size limits

## 📈 Results
| Model | Accuracy | Precision | Recall | F1-Score |
|---|---|---|---|---|
| Logistic Regression | 99% | 0.99 | 0.99 | 0.99 |
| LSTM | 98% | 0.98 | 0.98 | 0.98 |
| BERT | 99% | 0.99 | 0.99 | 0.99 |

## 🔍 Key Findings
- All three models performed exceptionally well on this dataset
- Logistic Regression surprisingly matched BERT at 99% accuracy, showing TF-IDF features are very effective for this task
- LSTM achieved 98% accuracy, slightly lower but still excellent
- BERT showed the most consistent precision across both classes
